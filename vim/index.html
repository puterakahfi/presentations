APA ITU VIM ?

Normal (/vɪm/; a contraction of Vi IMproved) is a clone of Bill Joy's vi text editor program for Unix. It was written by Bram Moolenaar based on source for a port of the Stevie editor to the Amiga and first released publicly in 1991.

VIM MODE

Normal : mode pengeditan atau navigasi, di mode ini anda tidak bisa melakukan pengetikan, untuk masuk ke mode ini tekan tombol ESC

Insert : Insert adalah mode untuk pengetikan, untuk masuk ke mode ini tekan tombol i

Visual/Visual-line : Mode penanda text untuk masuk ke mode visual menggunakan tombol v dan mode visual line menggunakan shift + v

Command-line : mode dimana kita bisa mengetikkan perintah-perintah, untuk menggunakan mode perintah gunakan :

Kunci penting mempelajari vim

Navigasi : navigasi merupakan keahlian yang sangat penting supaya anda bisa dengan leluasa mengeksplore workspace

Swich Mode : semakin baik anda bermain di swich mode, maka semakin cantik permainan and ai

Navigasi

navigasi ke atas : panah atas, atau 'k'
navigasi ke bawah : panah bawah, atau 'j'
navigasi ke kanan : gunakan panah kanan, atau 'l'
pindah ke kata berikutnya : gunakan 'w'
pindah ke kata sebelumnya : gunakan 'b'
pindah ke akhir kata : gunakan 'e'
pindah ke baris tertentu: nomorbaris di ikuti dengan 'G', contoh (10G) untuk loncat ke baris 10, atau bisa dengan cara masuk ke mode command line kemudian ketik :nomorbaris , contohnya :20

Penghapusan

menghapus 1 karakter ke kiri gunakan 'x'
menghapus 1 kata gunakan 'dw'
menghapus n kata gunakan 'jumlahkata+dw', misal menghapus dua kata ke kanan gunakan 2dw
menghapus 1 baris gunakan 'dd'
menghapus n baris kebawah gunakan 'jumlahbaris+dd', misal menghapus 2 baris kebawah gunakan 2dd
menghapus karakter mulai dari posisi awal kursor sampai akhir baris gunakan 'd$'
menghapus karakter mulai dari posisi kursos sampai awal baris gunakan 'd0'
menghapus seluruh baris gunakan ':%d' di mode command-line

Pencarian/pengubahan

mengubah 1 karakter gunakan 'r'
mengubah 1 kata gunakan 'cw'
mengubah 1 baris gunakan 'cc'
mencari 1 kata gunakan /kata di mode command-line/normal
